## üí¨ Slash Commands

| Command | What It Does |
|---|---|
| `/siggi` | Shows help (you'll need it) |
| `/siggi add [name]` | Spawns a new button. Congratulations. |
| `/siggi remove <id>` | Removes a button by ID. Goodbye, little friend. |
| `/siggi removeall` | Nuclear option. Removes everything. |
| `/siggi list` | Lists all your buttons in chat, in case you lost count |
| `/siggi rename <id> <name>` | Rename a button to something even worse |
| `/siggi icon <id> <path>` | Change a button's icon texture |
| `/siggi options` | Opens the options panel for civilized people |

You can also use `/sgl` if `/siggi` is too many characters for your weary fingers.
